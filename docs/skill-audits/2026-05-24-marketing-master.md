# Audit - Marketing Master

Fecha: 2026-05-24

## Hallazgos

- La skill tenia buenas rutas basicas: funnel, content, Meta Ads, email y SEO.
- Faltaba contrato SkillOps para no pisarse con `flyer-generator`, `video-script-generator`, `email-html-marketing`, `n8n-workflow-builder`, `documentar-notion` y Google Workspace.
- Faltaba capa de growth/lifecycle: CRM, calidad de lead, handoff comercial, retencion, reactivacion y medicion.
- Faltaba protocolo de busqueda externa antes de inventar campanas, plantillas o automatizaciones.

## Cambios

- Se agrego contrato de dominio y handoff.
- Se creo `references/growth-operating-system.md`.
- Se creo `references/research-and-external-alternatives.md`.
- Se amplio workflow y quality bar para demand creation/capture, lifecycle, datos, automatizacion y research.
- Se documento baseline en `docs/tool-verticals/marketing-potenciacion.md`.

## Decision

No crear skill paralela. `marketing-master` queda como cerebro de dominio; las skills productoras ejecutan piezas especificas.
