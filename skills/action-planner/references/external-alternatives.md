# External Alternatives For Action Plans

Use before designing a plan, class, roadmap, checklist, workshop, content structure or experience when existing templates/examples may accelerate quality.

## Search Targets

- Notion templates.
- Miro templates.
- GitHub planning templates.
- Course/session structures.
- Roadmaps and checklists.
- Reddit discussions about what worked/failed.
- Product management or project management templates.
- Visual examples of similar deliverables.

## Evaluation Criteria

| Criterion | Question |
| --- | --- |
| Clarity | Does the template make the idea easier to understand? |
| Fit | Does it match the user's audience and maturity? |
| Execution | Does it lead to a concrete deliverable? |
| Adaptability | Can it be adapted without becoming generic? |
| Cognitive load | Does it simplify or overwhelm? |
| Differentiation | Does it avoid looking like a generic template? |

## Required Output

Summarize:

- 3-7 patterns or examples found;
- what each contributes;
- what to adopt/adapt/avoid;
- recommended direction.

Do not copy templates literally. Extract structure and improve it for the user's goal.
