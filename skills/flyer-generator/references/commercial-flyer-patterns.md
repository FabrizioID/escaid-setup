# Commercial Flyer Patterns

Use this reference to choose the structure of a flyer, post, ad creative, carousel cover, event visual or promotional static piece.

This is not a new output format. It only improves the prompt and layout decision for the final image.

## Core Rule

A flyer has less time than a slide. It must win attention first, then make the next action obvious.

Default scan order:

1. Hook.
2. Visual tension or benefit.
3. Credibility/detail.
4. CTA.

If the viewer cannot understand the promise in 2 seconds, simplify.

## Pattern Picker

| Pattern | Best for | Layout structure | Prompt move |
| --- | --- | --- | --- |
| Big Hook Poster | Awareness, launch, event headline | Oversized headline, simple support visual, small CTA | Make the headline dominate and keep everything else secondary. |
| Pain Mirror | Problem-aware audience | Visual of current frustration plus sharp headline | Make the audience recognize their own pain instantly. |
| Before / After | Transformation offer | Split contrast, degraded vs improved state | Show the outcome difference before reading details. |
| Offer Stack | Promo, bundle, service package | Central offer block, benefits around it, CTA base | Make the value feel bundled and concrete. |
| Proof Badge | Authority, certification, credibility | Main claim plus badge/evidence cluster | Make proof more visible than decoration. |
| Event Hero | Summit, webinar, workshop, class | Event name, date, speaker/logo zone, atmosphere | Make it feel worth attending, not like a generic announcement. |
| Product Spotlight | Product/service reveal | Product/hero mockup central, 2-3 feature labels | Make the product or service the object of desire. |
| Countdown / Urgency | Deadline, limited spots, launch | Time marker large, CTA strong, minimal copy | Make time pressure visible without looking cheap. |
| Social Proof Wall | Testimonials, clients, results | Evidence grid behind one big takeaway | Show many trust signals supporting one claim. |
| Roadmap Mini | Program, course, process | 3-5 steps as compact path | Make the path feel achievable. |
| Myth Busting | Education + attention | Big myth crossed/fragmented, truth revealed | Make the correction visually satisfying. |
| Comparison Card | Choosing between options | Two-column contrast, highlighted winner | Make the preferred option obvious. |
| Premium Minimal | Institutional/high-ticket | Spacious layout, strong typography, subtle asset | Make restraint feel expensive. |
| Data Punch | Strong statistic or result | One large number, tiny context, visual proof | Make the number impossible to miss. |
| Community Call | Recruitment, cohort, membership | Faces/community energy, clear join signal | Make belonging feel immediate. |
| Founder/Expert Authority | Personal brand, expert post | Portrait/identity signal, bold claim, proof line | Make the person feel credible and sharp. |
| Framework Teaser | Thought leadership | Named framework, simple diagram, curiosity gap | Make the viewer want the full explanation. |
| Carousel Cover | First post of a sequence | Strong title, sequence cue, visual hook | Make the user want to swipe/read more. |

## Commercial Mechanisms

Choose one dominant mechanism per piece:

- Pain recognition.
- Aspirational transformation.
- Authority/proof.
- Curiosity gap.
- Urgency.
- Social belonging.
- Price/value clarity.
- New category creation.
- Myth correction.
- Simplicity/order.

## Format Defaults

- Instagram/feed square: 1:1, strong centered hook.
- Instagram/LinkedIn portrait: 4:5, headline upper third, visual middle, CTA lower third.
- Story/Reel cover: 9:16, headline upper/middle, CTA or brand lower.
- LinkedIn/document cover: 4:5 or 16:9 depending context, more sober typography.
- Event flyer: 4:5 or 9:16, date/time and brand must be scannable.

## CTA Rules

- One CTA only.
- Put CTA in a stable zone: bottom band, button-like marker, corner tag, or event info block.
- CTA copy must be action-oriented: `Reserva`, `Inscribete`, `Descarga`, `Agenda`, `Escribe`, `Conoce`, `Postula`.
- If the piece is editorial/top-of-funnel, CTA can be softer: `Desliza`, `Lee el caso`, `Mira el mapa`, `Guarda esta idea`.

## Anti-Patterns

- Flyer as mini document.
- Icon grid with no hook.
- Two headlines competing.
- CTA hidden in footer noise.
- Fake urgency when the offer is not urgent.
- Generic AI/tech glow when the audience needs a real-world cue.
- Too many logos, badges or data points.
- Every benefit in the same visual weight.

## Prompt Insert

Use this block inside the final prompt:

```text
Flyer pattern: [pattern].
Commercial mechanism: [pain / aspiration / proof / curiosity / urgency / belonging / value / category / myth / order].
Scan order: hook -> visual proof -> credibility/detail -> CTA.
Main hook text: [exact].
CTA text: [exact or none].
Layout: [zones and proportions].
Avoid: mini document, generic icon grid, hidden CTA, fake dashboard, decorative AI glow, duplicated message.
```
