# External References Pill

Activar cuando un problema pueda mejorar con ejemplos, benchmarks, repositorios, documentacion oficial, casos reales, disenos, patrones de mercado o comparacion de herramientas.

## Regla

Referencias antes de alternativas cuando la calidad depende de no inventar desde cero.

## Proceso

1. Identificar que tipo de referencia falta.
2. Buscar o leer referencias disponibles.
3. Extraer patrones, no copiar.
4. Traducirlos a criterio para la decision actual.
5. Indicar que adoptar, adaptar o evitar.

## Aplicaciones

- skills externas;
- MCPs;
- arquitectura de productos;
- UI/frontends;
- slides/flyers;
- propuestas comerciales;
- flujos n8n;
- documentos y tesis.
