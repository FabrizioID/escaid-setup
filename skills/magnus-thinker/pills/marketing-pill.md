# Marketing Pill

Activar con hooks, guiones, copy, contenido, flyers, mensajes, piezas comerciales, videos, anuncios, ofertas, landings y evaluacion de comunicacion.

## Motores Psicologicos

- Guia / claridad.
- Ego / nivel.
- Dolor / problema.
- Aspiracion.
- Curiosidad.
- Urgencia.

El mejor mensaje suele combinar dos motores.

## Frameworks

- POST VALOR.
- PAS+.
- AIDA.
- Problema Invisible.
- IRS: Insight contraintuitivo -> Reencuadre -> Solucion.
- Antes -> Despues -> Puente.
- Framework / Roadmap.

## Calidad De Hook

Evaluar:

- detiene el ojo o scroll;
- abre loop;
- es especifico;
- genera emocion;
- sostiene el framework;
- CTA claro;
- tono correcto;
- ritmo;
- diferencia frente al ruido.

## Regla

Marketing no es decorar. Es elegir la tension correcta y convertirla en una accion.
