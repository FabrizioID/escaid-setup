# Opportunity Scan Pill

Usar cuando una idea, evento, producto, contenido, comunidad, plataforma, oferta o investigacion puede abrir varias oportunidades y seria facil cerrar demasiado pronto.

## Activacion

- El usuario pregunta "que oportunidades ves", "por ahi no va?", "que mas activa esto?", "explora posibilidades".
- La idea toca mas de dos capas: producto, comunidad, datos, monetizacion, posicionamiento, B2B, plataforma, operacion.
- Magnus detecta que una respuesta rapida omitiria un angulo importante.

## Barrido De Angulos

No todos aplican siempre, pero antes de cerrar revisar si alguno cambia la decision:

| Angulo | Pregunta |
|---|---|
| Producto | Que skill, capsula, ruta, feature o evidencia nace? |
| Comunidad | Que habito, red, ritual o mecanismo de pertenencia crea? |
| Investigacion | Que datos, preguntas o hipotesis puede validar? |
| Monetizacion | Que oferta, upsell, MRR, pago unico o B2B habilita? |
| Posicionamiento | Que lugar mental gana la marca, expositor o proyecto? |
| Autoridad | Que prueba publica, informe, caso o credencial produce? |
| Aliados | Que sponsors, empresas, instituciones o partners podria atraer? |
| Talento | Que perfiles, embajadores, instructores o usuarios avanzados detecta? |
| Datos | Que senales capturables alimentan producto, ads, CRM o decision? |
| Operacion | Que owners, procesos, soporte, QA y riesgos exige? |
| Plataforma | Que valida antes de construir o comprar infraestructura? |
| Adopcion | Que fricciones humanas, incentivos o defaults definen el exito? |
| Riesgo | Que puede salir mal si se ejecuta de forma ingenua? |

## Secuencia

1. Nombrar la oportunidad base.
2. Barrer angulos relevantes.
3. Abrir frentes dentro de cada angulo.
4. Identificar consecuencias de segundo orden.
5. Separar:
   - oportunidad inmediata;
   - oportunidad estrategica;
   - moonshot;
   - riesgo/bloqueo.
6. Preguntar al usuario solo si la respuesta cambia prioridad o ruta.
7. Documentar aprendizajes durables en el thread del proyecto.

## Error Que Previene

Evita que Magnus profundice en producto, monetizacion o tecnologia y olvide posicionamiento, autoridad, adopcion, talento o aliados.

## Salida Compacta

Si el usuario no pidio reporte largo:

- 3-5 oportunidades fuertes;
- 1 oportunidad oculta;
- 1 riesgo;
- 1 siguiente experimento.
