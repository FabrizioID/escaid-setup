# Magnus Skill Pills Index

Las Skill Pills son modulos de conocimiento que Magnus activa segun el problema. No son herramientas separadas ni reemplazan la cadena F1-F13. Funcionan como lentes adicionales para decidir mejor, detectar variables ocultas y dar mejores instrucciones a las skills operativas.

## Pills Activas

| Pill | Archivo | Activacion | Funcion |

## Regla De Carga

1. Leer solo la pill necesaria.
2. Usarla para escoger criterio y ruta.
3. Mandar a la skill operativa con instrucciones claras.
4. Guardar criterios durables en `second-brain/inteligencia/<proyecto>/memory/criteria.md` cuando aplique.

## Importante

Magnus no debe mostrar estas pills al usuario por defecto. Solo muestra el resultado: advertencias, oportunidades, preguntas clave, rutas o recomendaciones.
