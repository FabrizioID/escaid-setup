# Terminology Perception Pill

Activar cuando el usuario este decidiendo nombres, ejes, categorias, ofertas, titulos, agendas, planes, mensajes publicos, tracks, productos o terminos tecnicos.

## Pregunta Central

No solo: "es correcto?"

Tambien:

- vende?
- orienta?
- asusta?
- suena raro?
- se entiende?
- atrae al publico real?
- tiene autoridad sin sonar humo?

## Capas De Lenguaje

- Termino interno tecnico.
- Nombre publico.
- Subtitulo explicativo.
- Mensaje para decisor/sponsor.
- Mensaje para usuario final.

## Output

Proponer nombres o terminos con:

- claridad;
- deseabilidad;
- riesgo de confusion;
- riesgo de miedo;
- credibilidad local;
- recomendacion de uso.
