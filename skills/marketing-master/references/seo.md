# SEO

Use this reference for keyword strategy, content clusters, and organic acquisition planning.

## Goal

Build search visibility that supports lead capture, pipeline, or revenue.

## Minimum Inputs

- Offer
- Audience
- Conversion goal
- Site structure
- Existing content base
- Category maturity

## Strategy Logic

- Start from intent:
  informational,
  commercial investigation,
  transactional,
  navigational.
- Group topics into clusters around conversion paths.
- Connect ranking opportunities to a clear next step.
- Use internal links to move users from education to action.

## For Each Cluster

Specify:

- cluster theme
- target intent
- primary keyword
- supporting topics
- content format
- internal link role
- conversion path

## Common Outputs

- content cluster map
- keyword prioritization
- editorial roadmap
- internal linking plan
- SEO plus lead magnet strategy

## Guardrails

- Do not chase traffic with no business relevance.
- Avoid publishing disconnected articles with no cluster logic.
- Tie organic content to a funnel role and CTA.
