## 3. Macrosecciones / frentes de trabajo

Use this section when the meeting covers more than one project, product, client, workstream, or strategic line. Each front must keep the full traditional meeting structure; do not flatten all agreements, risks, and questions into global lists.

Front-count rule:

- If there is only **one front**, use this template once and omit unnecessary repetition.
- If there are **2-3 fronts**, repeat the full front block for each front.
- If there are **4+ fronts**, repeat by front but consider batching the output to avoid losing detail. Ask the user before continuing if the response is becoming too long.

Default order is **front -> sections**, not **section -> fronts**.

Repeat this block for each front.
