# Frameworks And Formats

## Video Formats

Choose one primary format per script.

### Format 1: Support shots + voiceover

Best for tutorials, process explanations, before/after videos, product demos, and technical education.

Logic: narration guides; visuals prove or clarify.

### Format 2: Person on camera + support visuals

Best for personal authority, brand positioning, emotional connection, and trust-building.

Logic: the human face creates trust; visuals reinforce points.

### Format 3: Webinar or masterclass clips

Best for dense value, expertise demonstration, training excerpts, and B2B professional audiences.

Logic: the class format positions authority automatically.

### Format 4: Versus / comparison

Best for product differentiation, myths versus reality, before versus after, old method versus new method.

Logic: contrast creates immediate cognitive attention.

## Marketing Frameworks

Use 3 different frameworks across the 3 scripts.

### 1. POST VALOR

Structure: `Hook -> valuable teaching -> CTA`

Best for authority positioning and educational value.

Psychology: reciprocity plus credibility.

### 2. PAS+ (Problem -> Agitation -> Solution + Proof)

Structure: name the pain, amplify consequences, present solution, validate with evidence.

Best for direct sales, launches, and known objections.

Psychology: loss aversion plus social proof.

### 3. Optimal AIDA

Structure: attention, interest, desire, action.

Best for cold audiences and product discovery.

Psychology: controlled emotional progression.

### 4. Invisible Problem

Structure: reveal a problem the viewer did not know they had, explain why it matters, offer solution.

Best for saturated markets and audiences that think they do not need the offer.

Psychology: cognitive dissonance plus created urgency.

### 5. IRS (Insight -> Reframe -> Solution)

Structure: counterintuitive insight, reframe the viewer's perspective, deliver solution.

Best for differentiated positioning and belief-breaking.

Psychology: surprise plus epistemic authority.

### 6. Before -> After -> Bridge

Structure: painful current state, desired future state, bridge via product, method, or service.

Best for transformations, courses, and consulting services.

Psychology: hope, identification, aspiration.

### 7. Framework / Roadmap

Structure: present a step-by-step system where each step has logic and the full system becomes the value proposition.

Best for methodological authority, courses, trainers, and complex B2B.

Psychology: cognitive clarity plus trust in the system.

## Selection Logic

| Variable | Frameworks activated |
|---|---|
| Cold audience | Optimal AIDA, Invisible Problem, Before -> After -> Bridge |
| Warm audience | PAS+, IRS, POST VALOR |
| Hot audience | PAS+, Before -> After -> Bridge, Framework / Roadmap |
| Intent: educate | POST VALOR, Framework / Roadmap, IRS |
| Intent: sell | PAS+, Optimal AIDA, Before -> After -> Bridge |
| Intent: position authority | IRS, Framework / Roadmap, POST VALOR |
| Intent: viralize | Invisible Problem, IRS, Myth Busting angle |
| Professional B2B context | Framework / Roadmap, IRS, POST VALOR |
| Emotional B2C context | PAS+, Before -> After -> Bridge, Optimal AIDA |

If the 3 selected frameworks overlap too much, replace the least differentiated one with Invisible Problem or IRS.
