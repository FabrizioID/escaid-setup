# Magnus Handoff

The video script skill is the execution layer. Magnus is the strategic choice layer.

When a Magnus strategic input is present, follow it instead of independently choosing the main angle.

## Responsibility Split

Magnus chooses:

- viewer awareness state
- adoption barrier
- perception to break
- emotional/psychological lever
- dominant tension
- best hook family
- best framework family
- public-facing message angle
- what the viewer must feel in the first 3 seconds

The video script skill executes:

- validation brief
- hook variants
- script structure
- spoken lines
- visual/textual/auditory hook layers
- production direction
- CTA wording
- comparative table
- variants and rewrites

## If No Magnus Input Exists

Infer a provisional strategy, but mark it as provisional in the interpretation brief.

Do not over-optimize framework selection from the database alone. The database provides options; Magnus or the strategic analysis chooses what deserves to be used.

## Stage 1 With Magnus Input

When showing the interpretation brief, include one compact line:

```text
INPUT ESTRATEGICO MAGNUS
[resumen de barrera + palanca + angulo elegido]
```

Do not expose long internal reasoning unless the user asks.

## Hard Rule

If the user explicitly provides a Magnus-style strategic decision, do not replace it with a generic framework choice. Execute it.
