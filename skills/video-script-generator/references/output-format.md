# Output Format

## Script Structure

Each script must contain exactly these 11 sections:

```text
[HOOK SELECCIONADO]
[RETENCION INMEDIATA]
[CONTEXTO / SETUP]
[DESARROLLO / CUERPO]
[PUNTO DE INFLEXION]
[VALIDACION / PRUEBA]
[SOLUCION / METODO]
[TRANSFORMACION]
[CTA PRIMARIO]
[CTA SECUNDARIO]
[DIRECCION DE PRODUCCION]
```

## Viral Evaluation Criteria

Before presenting each script, evaluate internally:

1. Does the hook stop the scroll in the first 3 seconds?
2. Is there an open loop?
3. Is it specific?
4. Does it create emotion: surprise, identification, fear, hope, or tension?
5. Is the framework coherent from beginning to end?
6. Is the CTA direct and clear?
7. Is the tone correct for the audience?
8. Does the rhythm work when spoken out loud?
9. Is it meaningfully different from market noise?

If a script fails 3 or more criteria, rewrite it before presenting.

## Hard Rejection Checks

Reject and rewrite any script if:

- the hook could fit any generic business video
- the first 3 seconds do not include verbal, visual, and textual hooks
- the viewer does not know what reward they get by watching
- there is no actionable value, concrete example, or usable checklist
- the production direction lacks visual interruption or scene changes
- the CTA does not ask for a concrete action
- technical language is not translated into visible consequences

For warm audiences who already know the offer, the script must answer:

```text
Why should this matter now, if I already know the course exists?
```
